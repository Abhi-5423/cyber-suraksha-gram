# Admin Manual

Admins can access `/admin` after login.

Dashboard analytics show users, fraud checks, quiz participation, articles and top scam signals.

Articles can be added from `/admin/articles` in English, Hindi, Bhojpuri or Maithili.

Quiz questions can be added from `/admin/quiz`.

Scam reports can be reviewed from `/admin/reports`.

User roles can be changed from `/admin/users`. Roles are `citizen`, `volunteer` and `admin`.
